# 🛒 PDV_Java: Sistema de Frente de Caixa

Projeto desenvolvido em **Java** para automatizar o processo de vendas e controle de itens. Focado em aplicar conceitos sólidos de lógica de programação e POO.

## 🚀 Funcionalidades
- Interface funcional via Console (Terminal).
- Processamento de vendas e cálculo de valores.
- Estrutura organizada para fácil manutenção.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** Java 17+
- **IDE:** IntelliJ IDEA
- **Versionamento:** Git

## 📂 Como testar
1. Clone o repositório: `git clone https://github.com/Kaue150605/PDV_Java.git`
2. Importe o projeto no IntelliJ.
3. Execute o arquivo principal dentro da pasta `src`.
